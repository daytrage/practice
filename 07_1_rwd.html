viewportの設定

表示領域をスマホの画面幅に合わせるための設定

meta name="viewport" content="width=device-wid th, initial-scale=1.0"

注意：確認の時はブラウザのズーム率を100%にする。

CSS側

600px以上、800px以下

@media (min-width: 600px) and (max-width: 800px) {
body {
background: skyblue;
}
}
